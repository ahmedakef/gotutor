Privacy Policy — GoTutor browser extension

Last updated: April 7, 2026. Applies to the GoTutor extension published for Chromium-based browsers.

This policy describes how the GoTutor browser extension handles information. The extension adds shortcuts to open Go code in GoTutor from Go by Example and the Go Tour, and includes an optional popup to paste code.

1. Data the extension collects

The extension requests no browser permissions (such as storage, history, or broad site access). It does not run analytics, ads, or account login inside the extension. It does not read passwords or unrelated page content beyond what is needed to show its controls on the supported sites.

2. What happens when you use the extension

  • Go by Example: The extension adds links that open https://gotutor.dev/?id=… using playground snippet IDs already present in links on the page. It does not send your code anywhere by itself in this flow.
  • Go Tour and extension popup: When you choose to open GoTutor, the extension may send your editor or pasted Go source code to Go’s official share endpoint (https://go.dev/_/share) to obtain a snippet identifier, then open GoTutor with that ID in a new tab. That request is subject to Google’s / Go’s terms and privacy practices (see below).
  • GoTutor website (gotutor.dev): After you open a tab there, the web app may load shared code and send it to GoTutor servers to execute and visualize your program. The site may use its own cookies, analytics, and backend practices as described on that site. In short: your code can be processed on GoTutor’s servers and cached for performance, consistent with the in-app notice on gotutor.dev.

3. Third-party services

The extension interacts with services you explicitly trigger. Their policies govern those services:

  • go.dev / Go Playground share — operated by Google; see Google Privacy Policy and go.dev website policies.
  • gotutor.dev — operated by the GoTutor project; use the contact or policy information published on that site for questions about the web application.
  • gobyexample.com and pages under go.dev — subject to those sites’ own policies when you browse them.

4. Changes

We may update this policy when the extension or related services change. The “Last updated” date at the top will reflect the latest revision.

5. Contact

Questions about this extension policy: open an issue on the GoTutor GitHub repository or use the contact options listed for the project there.
